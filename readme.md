# minehunter

命令行中的扫雷游戏

Minesweeper game in command line

## Play

```bash
$ npx minehunter
```

或者安装到本地

Or install to local

```bash
$ npm i minehunter -g
$ minehunter
```

## CLI

使用 `minehunter --help` 查看帮助信息

Use `minehunter --help` see help message

```
$ minehunter --help

  Usage
	  $ minehunter

  Game Operation
    [↑/↓/←/→]  Move
    [space]    Open cell
    [f]        Flag
    [r]        Restart
    [q]        Quit
```
